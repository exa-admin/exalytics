# Product Definition

## Working Product Description

A Microsoft-native economic data analytics portal where users explore macroeconomic topics through interactive dashboards and written analysis.

## Working Name

Use `Exalytics` as the project and product name in design files, code, and documentation.

## Positioning

The product sits between a blog, a data portal, and a dashboard tool.

It should help users:

- Find economic topics quickly.
- Understand the latest database-backed values.
- Explore time-series charts.
- Read notes explaining what changed.
- Check sources, methodology, frequency, units, and latest update timestamps.

## Target Users

- Economists.
- Analysts.
- Students.
- Policy and market researchers.
- Readers who want plain-English analysis with data.
- Site owner/admin managing topics, notes, and data.

## Initial Topic Categories

- External Sector.
- Trade.
- Inflation.
- Growth.
- Fiscal.
- Monetary.
- Reserves.
- Labour.

## Initial Countries Or Regions

- India.
- United States.
- United Kingdom.
- Euro Area.
- Global.

## Main Navigation

Public navigation:

- Home.
- Topics.
- Analysis.
- Collaborate.

Admin/back-office navigation:

- Dashboard.
- Topics.
- Indicators.
- Notes.
- Data Imports.
- Refresh Runs.
- Settings.

Do not show admin/back-office pages in the public top navigation. Admin can be reached through a secure URL later.

## MVP Success Criteria

- A user can find Balance of Payments of India from the homepage.
- A user can open the topic dashboard and understand the latest current account position.
- A user can filter charts by date range and data type.
- A user can read analysis notes for the topic.
- An admin can draft and publish notes.
- The design clearly supports adding Trade Deficit of USA later.
