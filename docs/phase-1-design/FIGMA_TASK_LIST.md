## Review Checklist

- The first screen looks like a data portal, not a blog homepage.
- Balance of Payments India can be understood within 10 seconds.
- Latest database update timestamp is visible.
- Topic cards support future topics.
- Chart filters are clear.
- Mobile screens are usable.
- Admin notes editor has draft, preview, and publish states.
- Loading, empty, and error states exist.
- The design can be built in Blazor without unusual custom behavior.
