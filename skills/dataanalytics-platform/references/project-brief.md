# Project Brief

Use this brief when working on the analytics platform.

## Product

Microsoft-native economic analytics web application with interactive topic dashboards and CMS-style analysis notes.

Topic detail pages end with AI analyst notes generated from the topic's Azure SQL-backed data context. The AI note should be treated as a draft that a human reviews before publishing. The retrieval context includes observations, dimensions, source metadata, methodology, refresh runs, validation warnings, and previous analyst notes.

## First Dashboard

Balance of Payments of India.

Key metrics:

- Current account deficit or surplus.
- Goods exports.
- Goods imports.
- Services exports.
- Services imports.
- Net goods balance.
- Net services balance.
- Current account as percent of GDP.

## Future Dashboard Example

Trade Deficit of USA.

The second dashboard should prove the platform can add new topics mostly through data and configuration.
